# ImageFadeOut

```
Usage: ImageFadeOut [options] <OutputImage>

Arguments:
  OutputImage     The output image

Options:
  -i|--input      The input image
  -s|--start      FadeOut start point (default: 0)
  -d|--direction  FadeOut direction (Top, Bottom, Left, Right) (default: Bottom)
  -?|-h|--help    Show help information.

```
